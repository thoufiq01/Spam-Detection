# 📧 Spam Detection Model

## 🚀 Overview
This project is a machine learning-based spam detection system that classifies emails as spam or ham using NLP techniques.

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## ⚙️ Workflow
1. Data Cleaning
2. Label Encoding
3. Train-Test Split
4. TF-IDF Feature Extraction
5. Model Training
6. Prediction

## 📊 Result
Achieved high accuracy on training and test data.

## ▶️ Example
Input:
"I've been searching for the right words..."

Output:
Ham Mail ✅
